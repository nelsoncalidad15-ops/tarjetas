# Editor e impresor de tarjetas

Aplicación estática para crear, importar, organizar e imprimir tarjetas frente y dorso.

## Uso local

1. Instala dependencias:
   `npm install`
2. Inicia el proyecto:
   `npm run dev`

## Build

Para generar la versión publicable:

`npm run build`

El resultado queda en `dist/` y está preparado para hosting estático, incluido GitHub Pages.
